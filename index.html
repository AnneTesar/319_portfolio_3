Scalable Vector Graphics


The Basics


A shape can be drawn by specifying position and size. Color can be specified in a variety of ways, as can any style element.

<svg width="200" height="200" >
	<circle cx="50" cy="50" r="40" style="fill:rgb(150,0,255)" stroke="black" stroke-width="4" />
</svg >

We can create a variety of shapes, including custom polygons. For these, we specify a series of points to draw between.

<svg width="200" height="200" >
	<polygon points="15,15 30,80 80,150 150,45" style="fill:yellow;stroke:grey;stroke-width:2">
</svg >

We can also specify individual points for Paths. These use functions like Move To (M), Line To (L), Bezier Curve (q), and Close (Z) to draw shapes. These shapes can be filled in or not. Here we are also demonstrating that you can put multiple shapes into the same SVG tag. 

<svg width="200" height="200" >
	<path d="M 0 150 q 100 -200 200 0" stroke="blue" stroke-width="5" fill="none" />
	<path d="M 0 175 L 100 100 L 200 175 Z" stroke="blue" stroke-width="3" fill="rgb(100, 150, 200)" /> 
</svg >

The order that tags are in determines their order drawn on the page. The pink circle was there first, then the green circle was placed on top of it. The green circle has an opacity of 0.5 so you can still see the pink under it. 

<svg width="200" height="200" >
	<circle cx="50" cy="50" r="40" fill="pink" />
	<circle cx="75" cy="75" r="40" opacity="0.5" fill="green" /> 
</svg >

Basic Animation with SMIL

SMIL is inherent to SVGs.


This animate tag inside the rectangle tag applies the animation to that rectangle. We can specify things like starting and ending positions, duration of movement, and whether to repeat. 

<rect x="25" y="25" width="50" height="50" style="fill:green;fill-opacity:0.4;">
	<animate attributeName="x" from="25" to="150" dur="3s" repeatCount="indefinite" /> 
</rect>

There are other animation tags, such as Animate Transform, that allow us to change other properties. This time we also used an id to apply the animation, rather than having the animation wrapped in the ellipse tag. 

<ellipse id="ellipse" cx="50" cy="70" rx="50" ry="30" style="fill:orange">
	<animateTransform xlink:href="#ellipse" attributeName="transform" attributeType="XML" type="rotate" from="0 75 75" to="360 75 75" dur="3s" repeatCount="indefinite"/> 
</ellipse>

We can start animations on events like a click. Click the rectangle to change the opacity.

<rect x="25" y="25" width="50" height="50" style="fill:green;fill-opacity:0.4;">
	<animate attributeName="opacity" from="1" to="0" dur="3s" begin="click" /> 
</rect>

Basic Animation with CSS

Eventually SMIL is going to be deprecated. Can achieve same effects with CSS!


Text.

<rect x="25" y="25" width="50" height="50" style="fill:green;fill-opacity:0.4;">
	<animate attributeName="opacity" from="1" to="0" dur="3s" begin="click" /> 
</rect>

We can us CSS to change properties and create animations for SVGs.

<svg width="200" height="200" >
	<circle cx="50" cy="50" r="40" class="hue" />
</svg >

........

.hue {
  fill: red;
  animation: pulse 10s infinite;
}

@keyframes pulse {
  0%   { fill: #ff0000 }
  17%  { fill: #ffff00 }
  33%  { fill: #00ff00 }
  50%  { fill: #00ffff }
  67%  { fill: #0000ff }
  83%  { fill: #ff00ff }
  100% { fill: #ff0000 }
}
